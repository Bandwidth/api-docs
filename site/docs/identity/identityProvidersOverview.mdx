Bandwidth allows customers to specify their own Identity Provider (IdP) for user authentication. We currently support the SAML standard for configuring your IdP with Bandwidth and allow for you to do this programmatically with our API.

## API Specs

| API | Description |
|--|--|
| [Identity Provider (IdP) API](/apis/Identity) | API specification to configure your own Identity Provider within Bandwidth |
