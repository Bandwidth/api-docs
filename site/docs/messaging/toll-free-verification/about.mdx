The Bandwidth Toll-Free Verification API allows customers to submit verifications for toll-free phone numbers, listen to updates via webhook, and check on the verification status of phone numbers.

## Base API URL

`https://dashboard.bandwidth.com/api/v1/`

| Guide | Description |
|:------|:------------|
| [Submit a Request](./request.mdx) | How to submit a verification request |
| [Webhooks](./webhooks.mdx) | How to listen to status updates via webhook |

## Assumptions
* Familiarity with [Account API Credentials](../../../../account/credentials)
* You have read through the [toll-free verification handbook](https://support.bandwidth.com/hc/en-us/articles/1500006962542-Bandwidth-toll-free-verification-handbook-)
* Bandwidth has enabled toll-free verification API access for your user
* You understand that use of the API does not speed up the verification process
