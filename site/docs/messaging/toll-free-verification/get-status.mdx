This endpoint is used to get the verification status of a phone number.

### Example Request

```js
GET https://dashboard.bandwidth.com/api/v1/accounts/{accountId}/phoneNumbers/{phoneNumber}/tollFreeVerification HTTP/1.1
Authorization: Basic dXNlcm5hbWU6cGFzc3dvcmQ=
```
