This walks through how to programmatically submit a toll-free verification request. 

### Request Parameters
| Parameter                   | Type     | Required | Description                                                                                                                                                       |
|:----------------------------|:---------|:---------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| phoneNumbers                | `array`  | yes      | Array of strings, where each string is a toll-free phone number you are submitting a verification request for                                                     | 
| businessAddress             | `Object` | yes      | Object of address information of the business using the toll-free number(s)                                                                                       |
| businessAddress.name        | `string` | yes      | Name of the business using the toll-free number, <=500 characters                                                                                                 |
| businessAddress.addr1       | `string` | yes      | Address line 1 of the business using the toll-free number(s), <=500 characters                                                                                    |  
| businessAddress.addr2       | `string` | no       | Address line 2 of the business using the toll-free number(s), <=500 characters                                                                                    |
| businessAddress.city        | `string` | yes      | City of the business using the toll-free number(s), <=500 characters                                                                                              |
| businessAddress.state       | `string` | yes      | State of the business using the toll-free number(s), <=500 characters                                                                                             |
| businessAddress.zip         | `string` | yes      | ZIP Code of the business using the toll-free number(s), <=500 characters                                                                                          |
| businessAddress.url         | `string` | yes      | URL of the business using the toll-free number(s), <=500 characters                                                                                               |
| businessContact             | `Object` | yes      | Object containing contact information of an individual at the business using the toll-free number(s)                                                              |
| businessContact.firstName   | `string` | yes      | First name of the contact individual, <=500 characters                                                                                                            |
| businessContact.lastName    | `string` | yes      | Last name of the contact individual, <=500 characters                                                                                                             |
| businesContact.email        | `string` | yes      | Email of the contact individual, <=500 characters                                                                                                                 |
| businessContact.phoneNumber | `string` | yes      | Phone number of the contact individual, <=500 characters                                                                                                          |
| messageVolume               | `int`    | yes      | Estimated monthly volume of messages from the toll-free number(s)                                                                                                 |
| useCase                     | `string` | yes      | Category of use case for the toll-free number(s), <=500 characters                                                                                                |
| useCaseSummary              | `string` | yes      | General idea of the use case and customer, <=500 characters                                                                                                       |
| optInWorkflow               | `Object` | yes      | Object, summary of the workflow for opting into texts from the toll-free number(s)                                                                                |
| optInWorkflow.description   | `string` | yes      | Description of the opt-in workflow for the toll-free number(s), <=500 characters                                                                                  |
| optInWorkflow.imageUrls     | `array`  | yes      | Array of strings, where each string is a URL pointing to images showing the opt in workflow, each URL <=1000 characters                                           |
| productionMessageContent    | `string` | yes      | Example of message content from the toll-free number(s), <=1000 characters                                                                                        |
| additionalInformation       | `string` | no       | Any additional information you would like to provide, <=500 characters                                                                                            |
| isvReseller                 | `string` | no       | ISV name, if applicable                                                                                                                                           |
| webhookUrl                  | `string` | no       | Optional URL for Bandwidth to send status updates for the toll-free number(s) during the verification process, <=2000 characters (see [webhooks](./webhooks.mdx)) |

### Example

```js
POST https://dashboard.bandwidth.com/api/v1/accounts/{accountId}/tollFreeVerification HTTP/1.1
Content-Type: application/json; charset=utf-8
User-Agent: BandwidthAPI

{
  "phoneNumbers": ["+18335555555"],
  "businessAddress": {
    "name": "Bandwith Inc.",
    "addr1": "900 Main Campus Dr.",
    "addr2": "VCIII",
    "city": "Raleigh",
    "state": "NC",
    "zip": "27606",
    "url": "bandwidth.com"
  },
  "businessContact": {
    "firstName": "John",
    "lastName": "Doe",
    "email": "jdoe@bandwidth.com",
    "phoneNumber": "+19195555555"
  },
  "messageVolume": 10000,
  "useCase": "2FA",
  "productionMessageContent": "Your verification code is {code}",
  "optInWorkflow": {
    "description": "sign up for 2FA via our authenticator app",
    "imageUrls": [
      "bandwidth.com/images/optInToCustom2FAExample"
    ]
  },
  "additionalInformation": "Target customers will be our employees",
  "webhookUrl": "bandwidth.com/listen/for/updates"
}
```
